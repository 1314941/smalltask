LinkQueue类定义了一个单链表作为队列的数据结构，包含头节点和尾节点指针。该类提供了以下方法：

1. 构造函数：初始化头节点和尾节点指针。
2. 析构函数：释放队列中的所有节点。
3. empty()：检查队列是否为空。
4. push(T e)：将元素e入队。
5. pop(T &e)：将队列头部的元素赋值给e，并将其出队。
6. gethead(T &e)：将队列头部的元素赋值给e，不将其出队。
